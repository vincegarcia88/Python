O que é Python?

Python é uma linguagem de programação poderosa, de alto nível e fácil de aprender. Ela é amplamente utilizada em ciência de dados, desenvolvimento web, automação e muito mais!

Exemplo de Código

Veja como é fácil começar com Python:

# Mostra uma saudação
print("Olá, Mundo!")

# Soma dois números
a = 10
b = 5
print(a + b)
